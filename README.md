# Gaia playground

A simplified gaia repo to demonstrate and iterate on the new build workflow.

## Prerequisites

+ nodejs@0.12
+ npm@2
+ ninja

## Installation

```bash
git clone git@github.com:gaye/gaia-playground.git
cd gaia-playground
npm install
```

## Build

```bash
npm run build
```

## Clean

```bash
npm run clean
```
